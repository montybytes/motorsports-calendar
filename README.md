# MotorSport Calendar

This is a hobby project where I attempt to create a constantly updating calendar file that contains events from all the motorsports happening in a year. 

The file is updated daily but only in the event that an event is added, updated or removed from the schedule. As time progresses I will add more calendars to the project.
